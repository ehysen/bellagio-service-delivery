# Vision Prototype — Served Immediately, With Empathy

A single-file, clickable concept walkthrough of the **2030 vision**: *instant and personalized
access to public services, initiated by life events, delivered with empathy* — with government
carrying the burden of complexity.

It follows **Maya's first trimester** across six steps, showing two columns side by side:

- **What Maya experiences** — a phone with warm, plain-language messages, instant enrollments, and
  honest answers when a program is capped.
- **What the system carries** — the behind-the-scenes orchestration (life-event signal, consented
  data reuse, cross-program eligibility, event-driven cascades) that Maya never has to feel.

This is the artifact you fundraise and align states on — a concept prototype for vision alignment
only, **not** a real product. Programs, timings, names, and data are illustrative and fictional.

## Run it

It's a single static HTML file with no build step or dependencies:

```bash
open index.html          # macOS
# or just double-click index.html, or serve the directory:
python3 -m http.server   # then visit http://localhost:8000
```

Navigate with the **Next / Back** buttons, the dots, or the **← →** arrow keys.
